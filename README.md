# Chess

A chess game written in Python using Pygame.

## Requirements

- Python 3.12+ (or your version)
- Pygame

## Installation

Clone the repository:

```bash
git clone https://github.com/Faizan40404/Chess.git
cd Chess
```

Install Pygame:

```bash
pip install pygame
```

## Running the Game

```bash
python chess.py
```

## Project Structure

- `chess.py` - Main entry point
- `GUI.py` - Rendering and user interface
- `backend.py` - Game logic
